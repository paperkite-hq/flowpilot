# Examples

These examples demonstrate common patterns for building workflows with FlowPilot.

## Quick start

| Example | Description |
|---|---|
| [Hello World](/examples/hello-world) | Minimal workflow with two steps |
| [AI Summarizer](/examples/ai-summarizer) | Fetch a URL and summarize with an LLM |
| [Data Pipeline](/examples/data-pipeline) | Extract → transform → load pattern |
| [Webhook Handler](/examples/webhook-handler) | Process incoming webhooks with validation |

## Running examples

All examples live in the `examples/` directory and can be run directly:

```bash
bun run examples/hello-world.ts
```

Or using the CLI:

```bash
bun run flowpilot run examples/hello-world.ts
```
